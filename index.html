Graphic Novel Subtitling

A web-based application built using Reactand Node.js. When a video and subtitle file are uploaded, keyframesare extracted and the subtitle file is parsed with text assigned tocorresponding keyframes based on timestamp information. Framesare then styled according to user preferences.

Try it yourself: graphic-novel-subtitling.com
